# yuliao_pyqt

自制语料分析小工具，使用 PyQt5 制作 UI

## 使用方法

- Requirements
    - Python>=3.6
    - pyqt5>=5.12
    - pandas
    
- 将所需语料以 txt 格式存储，编码须为 GB2312。
- 有两种方式标识对话
    - 在文件开头标识，如
    ```
    1-法国
    2-中国
    
    1你好，我叫亚当。
    2你好，我叫张明。
    ```  
    - 在每段对话行首标识，如
    ```
    1（法国）你好，我叫亚当。
    2（中国）你好，我叫张明。
    ```
    - 在国别后可加上姓名，便于统计，如`1-法国-亚当`
- 命令行启动 `python main.py`

## 功能介绍

- 搜索语料库出现词语
- 包括国家/地区、剔除国家/地区
- 修改语料库
- 将搜索结果导出至 excel 文件
- 按国家统计字数
